# Navigation Links Hover Animation with CSS Tooltip

Um componente de interface textual moderno, limpo e responsivo focado em microinterações avançadas para hiperlinks, desenvolvido exclusivamente com HTML5 e CSS3. O projeto apresenta um bloco de apresentação biográfica onde links específicos (como canais, websites e redes sociais) acionam balões de diálogo informativos (tooltips) animados ao passar o mouse.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do contêiner de texto e marcação de hiperlinks (`<a>`).
* CSS3: Estilização completa, uso de bordas arredondadas, sombras projetadas e gerenciamento de fontes.
* Animações em CSS: Uso de pseudo-elementos (`::before` e `::after`) combinados com propriedades de transição (`transition`), opacidade e transformação de escala/posicionamento.

## Funcionalidades e Técnicas Aplicadas

* Tooltips em CSS Puro: Balões de informação ocultos que surgem de forma fluida acima dos links (como CodingLab, Website, Facebook e Instagram) sem necessidade de JavaScript.
* Microinterações no Hover: Efeitos visuais dinâmicos que gerenciam a suavidade de entrada (fade-in e slide-up) do tooltip quando o utilizador posiciona o cursor sobre os termos destacados.
* Formas Geométricas com Pseudo-elementos: Uso de truques de borda para desenhar a seta indicadora na base do balão de diálogo diretamente via CSS.
* Layout Minimalista e Confortável: Organização tipográfica equilibrada com espaçamento entre linhas adequado, disposta em um card branco com cantos suavizados e sombra sutil.
